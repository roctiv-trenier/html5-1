Elementos de bloco: 

Eles consomem toda a largura disponível, independentemente de sua suficiência. Eles sempre 
começam em uma nova linha e têm margens superiores e inferiores. Não contém outros elementos 
ao seu lado.


Exemplos de elementos de bloco:

<h1>-<h6>: Este elemento é usado para incluir títulos de diferentes tamanhos que variam de 1 a 6.

<div>: Esta é uma tag de contêiner e é usada para fazer divisões separadas de conteúdo na página web.

<hr>: Esta é uma tag vazia e é usada para separar o conteúdo por linhas horizontais.

<li>: Esta tag é usada para incluir itens de lista de uma lista encomendada

ELEMENTOS DE BLOCO SEMântico HTML 5:

<header>:Esta tag é usada para incluir todas as principais coisas da página da web, 
como navbar, logotipos e título da página web.

<nav>:Esta tag ajuda a navegar através de diferentes seções, incluindo diferentes 
blocos de hiperlinks na página web.

<article>:Esta tag é usada para incluir diferentes artigos independentes na página web.
<aside>:Esta tag é usada para mencionar detalhes do conteúdo principal à parte.

Elementos inline: 

Elementos inline ocupam apenas largura suficiente que é suficiente para ele 
e permite outros elementos ao seu lado que são inline. Os elementos inline não começam a 
partir de uma nova linha e não têm margens superiores e inferiores como os elementos de bloco
têm.


Exemplos de elementos inline:

<a>:Esta tag é usada para incluir hiperlinks na página web.
<br>:Esta tag é usada para mencionar quebras de linha na página da web onde for necessário.
<script>: Esta tag é usada para incluir códigos JavaScript externos e internos.
<input>:Esta tag é usada para tirar informações dos usuários e é usada principalmente em formulários.
<img>:Esta tag é usada para incluir diferentes imagens na página web para adicionar beleza à página web.
<span>:Este é um recipiente interno que leva apenas o espaço necessário.
<b>:Esta tag é usada em lugares onde o texto em negrito é necessário.
